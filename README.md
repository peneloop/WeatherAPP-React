## 🌤 Aplicativo de Previsão do Tempo feito com React e TailwindCSS

Nesse projeto tive o objetivo de melhorar minhas habilidades no framework React, enquanto tive um contato maior com a utilização de APIs e suas ferramentas, OpenWeather, em conjunto da chamada assíncrona no JavaScript e da biblioteca Axios para facilitar as requisições para o servidor da API.

Aproveitei para fazer melhorias visuais, criando minha própria estilização com TailwindCSS v4 para melhorar a performance da aplicação.
